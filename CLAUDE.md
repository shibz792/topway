# Topway Implementation Rules

- Objective: convert international employers requesting Sri Lankan workers.
- Primary CTA: Request Workers. Candidate actions remain secondary.
- Keep pages calm: one purpose, one dominant action, no repeated proof or CTA sections.
- Use semantic CSS tokens and existing reusable components.
- Mobile first: 44px targets, no overflow, no desktop-only interactions.
- Motion explains recruitment flow; support reduced motion.
- Never invent vacancies, active markets, clients, statistics, guarantees or licence details.
- Meet WCAG 2.2 AA and preserve server components where possible.
- Keep initial JavaScript low; use Next Image and `next/font`.
- Verify with `npm run typecheck`, `npm run build`, route checks and responsive review.
