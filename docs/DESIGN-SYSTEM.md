# Design System

## Character
Authoritative, calm, structured and human. Use Topway teal and charcoal with warm neutral surfaces and restrained gold compliance accents.

## Foundations
- Display: Instrument Sans via `next/font`.
- Body: Inter via `next/font`.
- Fluid type: 16px body, 18-24px cards, 32-48px sections, 42-68px hero.
- Containers: 1180px content, 760px reading width.
- Spacing: 4, 8, 12, 16, 24, 32, 48, 72, 96.
- Radius: 6, 10, 16, full.
- Shadows: subtle surface and elevated only.

## Hierarchy
- Primary button: Request Workers only.
- Secondary button: process, licence or contact.
- Text link: lower-priority exploration.
- Cards are used only for categories, markets and team profiles.

## Tokens
All new work uses semantic CSS variables for background, surfaces, foreground, muted text, primary, accent, state, border, focus, spacing, radius, shadow, motion and z-index.
