# Responsive Rules

- Design from 320px upward; no horizontal overflow.
- 44px minimum interactive targets.
- Hero becomes content-first; visual follows and simplifies below 760px.
- Desktop grids use intrinsic `auto-fit/minmax`; mobile becomes one column.
- Process is horizontal on desktop and vertical on mobile.
- Forms are one column below 700px.
- Mobile menu is a full-height accessible overlay with safe-area padding.
- Sticky mobile actions contain Call, WhatsApp and Request Workers only.
- Sticky elements must reserve content space and respect safe-area insets.
- Verify at 320, 360, 375, 390, 430, 768, 1024, 1280, 1440 and 1920px.
