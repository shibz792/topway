# Motion System

## Micro
Buttons, links and cards: 160-220ms transform/opacity feedback.

## Section
Use restrained fade/translate reveals only where interaction code already exists. Reading never depends on animation.

## Signature
1. Hero recruitment flow: candidate profiles progress into a verified shortlist.
2. Recruitment process: a transform-based route line shows progression.

Avoid decorative motion and animation of layout dimensions. Reduced motion disables all non-essential movement.
